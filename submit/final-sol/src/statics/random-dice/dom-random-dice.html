DOM Random Dice

Animate the two dice shown below:

  • Roll the two dice every second so that the value of each dice changes randomly between 1 through 6.
  • Increment the # of rolls counter on each roll.

Hints

  • Your code should maintain the values of the two dice and # of rolls in state variables.
  • Start an interval timer.
  • For every tick of the timer, roll the dice to update the dice state variables, increment the # of rolls and reflect the value of the state variables into the HTML.
  • For rolling a die, use Math.random() and Math.trunc(). Note that Math.random() returns a random number x such that 0 <= x < 1. This will need to be transformed to an integer between 1 through 6.
