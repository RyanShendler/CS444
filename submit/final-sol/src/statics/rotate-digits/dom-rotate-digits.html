Hints

  • Note that each <td> element has a data-index attribute which gives the index of each cell in the sequence of the rotation. Hence your code needs to simply paint each character of a rotation of the digits string "12345678" into the appropriate cell.
  • Your code can maintain the current rotation of the digits string in a state variable.
  • Start an interval timer.
  • For every tick of the timer, rotate the digits string maintained in the state variable, and paint each character of the rotated digits string into the appropriate cell in the HTML.
  • Rotate the digits string right by using String.slice().
