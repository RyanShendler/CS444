DOM Rotating Text

Set up a rotating text animation which works as follows:

  • Whenever the Text input widget is blurred, the contents of what follows the Rotating Text label should reflect the contents of the input widget.
  • The contents of what follows the Rotating Text label should be rotated left every 1 second.

The source code for this page is available in src/statics/rotate-text/dom-rotate-text.html.

Rotating Text

Hints

  • Your code can maintain the current rotation of the text string in a state variable.
  • A blur handler for the input widget can be used to set the text state variable to the value of the input widget. The value of the text state variable should also be reflected into the <span> element following the Rotating Text label.
  • Start an interval timer.
  • For every tick of the timer, rotate the text string maintained in the state variable left and reflect its value into the <span> element following the Rotating Text label.
  • Rotate the text string left by using String.slice().
