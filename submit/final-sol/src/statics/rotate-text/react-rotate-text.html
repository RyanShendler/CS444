React Rotating Text

Animate rotating text with functionality identical to that given in the DOM version, but using React hooks.

  • Write a React component which renders the input widget and rotating text animation.
  • The provided code does not generate the content at all. The component will need to provide the rendering.
  • Your component should allow changing the animation period using the component's props with a default value of 1 second.
  • This page loads all necessary React libraries as well as a babel transformer. Specifically, the React and ReactDOM objects are available for use by your code.

The source code for this page is available in src/statics/rotate-text/react-rotate-text.html.

Hints

  • Start out your component by simply having it render JSX corresponding to the HTML provided in the DOM version.
  • Your code should maintain the values of the current rotation of the text string in a state variable.
  • Create a blur handler for the input widget which sets the text state variable to the provided value.
  • Start an interval timer in an effect which is run only when the component is first mounted. To ensure that the timer gets cleaned up, keep its id in an additional state variable.
  • For every tick of the timer, rotate the text string left. Watch out for stale closures.
  • You can use the same code for rotating left as you did in your DOM version.
